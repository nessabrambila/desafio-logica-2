# Projeto – Calculadora de Partidas Rankeadas

Projeto desenvolvido para prática de lógica de programação em **JavaScript**.

## Objetivo
Criar uma função que recebe a quantidade de vitórias e derrotas de um jogador, calcula o saldo de partidas e define o nível do jogador utilizando estruturas de decisão.

## Conceitos Utilizados
- Variáveis  
- Operadores  
- Funções  
- Estruturas de decisão  

## Classificação por Vitórias
| Vitórias       | Nível      |
|----------------|------------|
| Menor ou igual a 10   | Ferro      |
| 11 – 20        | Bronze     |
| 21 – 50        | Prata      |
| 51 – 80        | Ouro       |
| 81 – 90        | Diamante   |
| 91 – 100       | Lendário   |
| Maior que 100  | Imortal    |

## Saída Esperada
"O Herói tem de saldo de **{saldoVitorias}** está no nível de **{nivel}**"
